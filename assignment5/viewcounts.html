View Counts

   This table tells you about the view counts of each show. Lauched period refers to the total number of days since the show's release date. Accelerated view counts refers to the total amount of view counts up to the date of 10/26/2018. Average view counts = accerated view counts/lauched period, which becomes the average view count per day. Highest daily view counts refers to the highest single day view counts in the past month.

   Now the records are displayed in the quantity order of accelerated view counts. You may change it by using the "Sort" button. And use the "Group" button to see which platform or genre gains the highest view counts.
